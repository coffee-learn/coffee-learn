Coffee Learn
============

coffee learn is a machine learning library written in coffeescript.

# Documentation

http://coffee-learn.github.io/coffee-learn/

# Installation

### Using on browser

```html
<script src="/path/to/cflearn.min.js"></script>
```

### Install with npm
```bash
$ npm install coffee-learn
```

### Install with Bower
```bash
$ bower install coffee-learn
```
